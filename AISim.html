Enhancement Engine

PROMETHEUS v7.0

The world's most advanced prompt optimization system. Research-backed, cognitively-optimized, and validated for 40-70% improvement.

Zone 1: Primacy (First 20%)

  • Role/Identity definition
  • Critical constraints (MUST NOT)
  • Core task definition
  • Security-critical instructions

Zone 2: Context (Middle 60%)

  • Background information
  • Methodology specification
  • Examples (if format non-obvious)
  • Specifications and parameters

Zone 3: Recency (Final 20%)

  • Output format specification
  • Final constraint reminders
  • Success criteria
  • Activation phrase ("Begin now")
